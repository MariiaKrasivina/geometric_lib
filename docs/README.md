# Math formulas
## Area
- Circle: S = πR²
**Здесь ищем площадь круга**
- Rectangle: S = ab
**Здесь ищем площадь прямоугольника**
- Square: S = a²
**Здесь ищем площадь квадрата**

## Perimeter
- Circle: P = 2πR
  **Здесь ищем периметр круга**
- Rectangle: P = 2a + 2b
  **Здесь ищем периметр прямоугольника**
- Square: P = 4a
  **Здесь ищем периметр квадрата**
