# Math formulas
## Area

**Площадь круга**
- Circle: S = πR²
**Площадь прямоугольника**
- Rectangle: S = ab
**Площадь квадрата**
- Square: S = a²


## Perimeter

**Периметр круга**
- Circle: P = 2πR
**Периметр прямоугольника**
- Rectangle: P = 2a + 2b
**Периметр кавдрата**
- Square: P = 4a
